# Sephora (sephora)

Sephora is a French multinational chain of personal care and beauty stores and a subsidiary of LVMH, operating 2,700+ locations worldwide and a large ecommerce footprint. Sephora does not publish a public developer API and does not maintain a public GitHub organization. Affiliate partnerships are brokered through third-party networks such as Rakuten Advertising.

**URL:** [https://www.sephora.com/](https://www.sephora.com/)
